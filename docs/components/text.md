# text

Text blocks modifiers.

```less
@import 'typro/components/text';
```

```html
<p class="text--left">
	Lorem ipsum.
</p>

<p class="text--center">
	Lorem ipsum.
</p>

<p class="text--right">
	Lorem ipsum.
</p>

<span class="text--muted">Lorem ipsum</span>
<span class="text--bold">Lorem ipsum</span>
<span class="text--strike">Lorem ipsum</span>
```


## Variables

* `@typro-text-muted`
