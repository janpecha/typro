Test case file name
===================

Minifier\<class>.<method()>.<area>.<number>.phpt

Minifier\TyproMinifier.phpt                - tests for a class's basic behaviour
Minifier\TyproMinifier.minify().phpt      - tests for a method's basic behaviour
Minifier\TyproMinifier.minify().inc       - common code for more test cases
Minifier\TyproMinifier.minify().expect    - expected raw output
Minifier\TyproMinifier.minify().area.phpt - tests for a specified area of class/method

- areas: basic, error, bug#123
- numbers have three digits
